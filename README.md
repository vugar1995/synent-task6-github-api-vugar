# Synent Task 6 - GitHub API Integration Project

A Fetch API project that retrieves and displays GitHub profile and repository data dynamically.

## Requirement Coverage
- JavaScript Fetch API
- GitHub public API
- Dynamic profile and repository rendering
- Loading state
- Error handling

## How to Run
Open `index.html` in a browser and search for a GitHub username.
